HTML

Hyper Text Markup Language
*HyperText = 태그를 의미한다

- 태그는 대부분 두쌍씩 사용한다
- 태그는 중첩하여 사용이 가능하다 (부모태그&자식태그)
- 앞에 있는 태그(<>)는 열리는 태그, 뒤에 있는 태그(</>)는 닫히는 태그

관련 사이트 모음

  1. MDN
    : 웹 관련 정보를 가장 빠르게 확인할 수 있다
  2. validator
    : 오류 검사할 수 있는 사이트

태그 모음

Main root

<html>
* 이 문서가 html 문서임을 알림

Document metadata

* 페이지 정보를 포함하는 태그들
  1. <head>
    * 본문 관련 설명을 입력
    * <title>, <meta>와 같은 아래 다른 코드들을 넣는다
  2. <meta>
    * 해당 문서의 메타데이터를 정의
    ex) <meta charset="utf-8">
    ex) <meta name="viewport" content="width=device-width">
  3. <title>
    * 웹페이지의 제목 지정
    * 브라우저에서 웹페이지를 검색할 때 사용한다
    * 지정하지 않으면 파일명을 그대로 사용
  4. <base>
    * 해당 문서의 베이스가 되는 URL을 사용할 경우 이용하는 태그
    * 문서에서 한 번만 사용할 수 있다
  5. <link>
    * 해당 문서와 관련된 문서 및 외부 소스를 링크하는 태그
    * css나 외부 리소스 아이콘을 사용하는 등의 경우에 사용
    ex) <link rel="stylesheet" href="css/master.css">
    ex) <link rel="icon" href="favicon.ico">
  6. <style>
    * 문서에 적용되는 스타일 정보를 포함하는 태그
    * 내용에 CSS가 적힌다

기본 구조 태그

  1. <body>
    - 본문 내용 관련 코드를 넣는 태그
    - 사용자에게 보여지는 내용을 작성한다

위 태그들을 사용하여 아래와 같은 형태의 html 문서의 기본 구조를 만들 수 있다
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width">
    <title> @@@ </title>
  </head>
  <body>
    somethings
  </body>
</html>

ITEM TAGS

- 사용자에게 보여지는 것들을 표현하기 위한 태그
- block/inline level로 나누어진다
  * block : 한 줄에 하나
  * inline : 공간이 허용되면 다른 태그 옆에 배치
- content : tag 내부에 들어간 내용
- Element : Opening Tag - Content - Closing Tag까지
- Attributes(속성) : opening tag에서 태그에 대한 자세한 속성값을 설정
  1. <a>
    - 링크(anchor)
    - ex) <a href="링크 주소" target="_blank" title="~~">
    ㄴ href(HyperText Reference)
    ㄴ target="_blank" : 링크 클릭 시, 새 창에서 페이지가 열리게 하는 속성
    ㄴ title="~~" : 커서 올렸을 때, 링크가 어떤 내용을 담고 있는지 툴팁으로 보여주는 기능
    <op.gg>
    (생코)
  2. <button>
  3. <input>
    - 입력 형식을 구현하기 위한 태그
    - 보통 lable태그와 함께 사용 - type 속성을 사용하여 여러 입력 방식을 사용한다










  4. <label>
  5. <img>
    - 이미지 태그
    - src(sorce) 속성을 같이 사용해야한다
    - 인터넷 이미지 주소나 로컬 이미지 주소를 넣는다
    ex) <img src="https://s3-ap-northeast-2.amazonaws.com/opentutorials-user-file/module/3135/7648.png">
    - width 속성을 사용하여 숫자나 %로 크기를 조정할 수 있다
  6. <video>
  7. <audio>
  8. <map>
  9. <canvas>
  10. <table>

문장 관련 태그

  1. <p>
    - 단락 태그
    - <p> 태그와 역할이 같지만 단락을 표현할 때 사용하면 더 좋다
    - 웹페이지를 정보로써 더 의미있게 전달하기 때문이다
    - 단락과 단락 사이 간격이 고정되어 있어 시각적 자유도가 떨어지나, css를 사용하여 조정할 수 있다
    ex) <p style="margin-top:45px">
  2. <strong / b>
    - 두꺼운 글씨 - inline 태그
  3. <span>
    - inline 태그
  4. <u>
    - underline
    - 밑줄
  5. <h1>~<h6>
    - 제목 태그
    - 숫자가 작을수록 글씨가 커진다
  6. <br>
    - 줄바꿈 태그
    - 쌍 없이 문장 끝에 하나만 사용한다
  7. <li>
    - 리스트 태그 - <ul>(unordered list), <ol>(ordered list)를 부모로 사용한다
    ex)
    <ul>
      <li>
       <something 1
       <something 2
      </li>
    </ul>

CSS 관련 태그

  1. <style>
    - 내부 코드가 css 문법임을 나타내는 태그
    - <head> 안에서 사용한다
  2. <div>
    - 의미는 없지만 디자인 용도로 태그가 필요할 때 사용한다
    - block 단위 or inline 단위 태그이다
  3. <link>
    - 이 웹페이지가 어떤 CSS 파일과 연결되어 있는지 표현하는 태그
    - rel 속성과, href 속성을 사용한다

JavaScript 관련 태그

  1. <script>
    - 내부 코드가 JS 문법임을 나타내는 태그
    - <body> 안에서 사용한다
