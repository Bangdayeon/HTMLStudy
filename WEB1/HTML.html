HTML

Hyper Text Markup Language
*HyperText = 태그를 의미한다

- 태그는 대부분 두쌍씩 사용한다
- 태그는 중첩하여 사용이 가능하다 (부모태그&자식태그)
- 앞에 있는 태그(<>)는 열리는 태그, 뒤에 있는 태그(</>)는 닫히는 태그

관련 사이트 모음

  1. MDN
    : 웹 관련 정보를 가장 빠르게 확인할 수 있다
  2. validator
    : 오류 검사할 수 있는 사이트

태그 모음

기본 구조 태그

  1. <!doctype html>
    - 이 웹페이지가 HTML로 제작되었다는 것을 표현
  2. <html>
    - body 태그와 head 태그를 감싸는 하나의 태그
  3. <head>
    - 본문 설명 관련 코드를 넣는 태그
    - <title>, <meta>와 같은 코드를 넣는다
  4. <meta>
    - 해당 문서에 대한 메타데이터를 정의할 때 사용한다
    ex)
    <meta charset="utf-8">
    : 모든 언어를 지원할 수 있음
    <meta name="viewport" content="width=device-width">
    : 컨텐츠 크기를 디바이스 크기에 맞춤
    - 본문이 아닌 본문을 설명하는 역할
  5. <title>
    - 웹페이지의 제목 지정
    - 지정하지 않으면 파일명이 그대로 탭에 쓰인다
    - 검색엔진이 웹페이지를 분석할 때 중요하게 쓰이는 태그
    - 브라우저에서 웹페이지를 검색할 때 사용한다
    - 본문이 아닌 본문을 설명하는 역할
  6. <body>
    - 본문 내용 관련 코드를 넣는 태그
    - 사용자에게 보여지는 내용을 작성한다

위 태그들을 사용하여 아래와 같은 형태의 html 문서의 기본 구조를 만들 수 있다
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width">
    <title> @@@ </title>
  </head>
  <body>
    somethings
  </body>
</html>

기타 태그

  1. <strong>
    - 두꺼운 글씨
  2. <u>
    - underline
    - 밑줄
  3. <h1>~<h6>
    - 제목 태그
    - 숫자가 작을수록 글씨가 커진다
  4. <br>
    - 줄바꿈 태그
    - 쌍 없이 문장 끝에 하나만 사용한다
  5. <p>
    - 단락 태그
    - <p> 태그와 역할이 같지만 단락을 표현할 때 사용하면 더 좋다
    - 웹페이지를 정보로써 더 의미있게 전달하기 때문이다
    - 단락과 단락 사이 간격이 고정되어 있어 시각적 자유도가 떨어지나, css를 사용하여 조정할 수 있다
    ex) <p style="margin-top:45px">
  6. <img>
    - 이미지 태그
    - src(sorce) 속성을 같이 사용해야한다
    - 인터넷 이미지 주소나 로컬 이미지 주소를 넣는다
    ex) <img src="https://s3-ap-northeast-2.amazonaws.com/opentutorials-user-file/module/3135/7648.png">
    - width 속성을 사용하여 숫자나 %로 크기를 조정할 수 있다
  7. <li>
    - 리스트 태그 - <ul>(unordered list), <ol>(ordered list)와 함께 사용한다
    ex)
    <ul>
      <li>
       <something 1
       <something 2
      </li>
    </ul>
  8. <a>
    - 링크 태그 (anchor)
    - ex) <a href="링크 주소" target="_blank" title="~~"> ㄴ href(HyperText Reference)
    ㄴ target="_blank" : 링크 클릭 시, 새 창에서 페이지가 열리게 하는 속성
    ㄴ title="~~" : 커서 올렸을 때, 링크가 어떤 내용을 담고 있는지 툴팁으로 보여주는 기능
    <op.gg>
    (생코)

CSS 관련 태그

  1. <style>
    - 내부 코드가 css 문법임을 나타내는 태그
    - <head> 안에서 사용한다
  2. <div>
    - 의미는 없지만 디자인 용도로 태그가 필요할 때 사용한다
    - block 단위 or inline 단위 태그이다
  3. <link>
    - 이 웹페이지가 어떤 CSS 파일과 연결되어 있는지 표현하는 태그
    - rel 속성과, href 속성을 사용한다

JavaScript 관련 태그

  1. <script>
    - 내부 코드가 JS 문법임을 나타내는 태그
    - <body> 안에서 사용한다
  2. <input>
    - 입력 형식을 구현하기 위한 태그
    - type 속성을 사용하여 여러 입력 방식을 사용한다
    + type의 속성값으로는 아래 내용들이 올 수 있다
    ** text : 텍스트를 입력하는 창 생성
    ** password : 비밀번호를 입력하는 창 생성
    ** radio : 라디오 버튼 생성
    ** checkbox : 체크박스 생성
    ** file : 파일 이름 입력 창 생성
    ** image : 이미지를 전송 버튼으로 제작
    ** hidden : 사용자에게 보이지 않지만 서버로 전송
    ** submit : 서버로 제출/전송하는 버튼을 제작
    ** button : 일반 버튼을 생성
