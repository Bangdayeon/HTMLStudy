HyperText = 태그를 의미

- 태그는 대부분 두쌍씩 사용한다
- 태그는 중첩하여 사용이 가능하다
- 앞에 있는 태그(<>)는 열리는 태그, 뒤에 있는 태그(</>)는 닫히는 태그

태그들 모음

구조 관련 태그

  1. <!doctype html>
    - 이 웹페이지가 HTML로 제작되었다는 것을 표현
  2. <html>
    - body 태그와 head 태그를 감싸는 하나의 태그
  3. <head>
    - 본문 설명 관련 코드를 넣는 태그
    - <title>, <meta>와 같은 코드를 넣는다
  4. <body>
    - 본문 내용 관련 코드를 넣는 태그
  5. <title>
    - 웹페이지의 제목 지정
    - 지정하지 않으면 파일명이 그대로 탭에 쓰인다
    - 검색엔진이 웹페이지를 분석할 때 중요하게 쓰이는 태그
    - 본문이 아닌 본문을 설명하는 역할
  6. <meta>
    - 해당 문서에 대한 메타데이터를 정의할 때 사용한다
    ex) charset 속성을 사용하여 utf-8과 같은 문서 형식 정의
    - 본문이 아닌 본문을 설명하는 역할

기타 태그

  1. <strong>
    - 두꺼운 글씨
  2. <u>
    - underline
    - 밑줄
  3. <h1>~<h6>
    - 제목 태그
    - 숫자가 작을수록 글씨가 커진다
  4. <br>
    - 줄바꿈 태그
    - 쌍 없이 문장 끝에 하나만 사용한다
  5. <p>
    - 단락 태그
    - <p> 태그와 역할이 같지만 단락을 표현할 때 사용하면 더 좋다
    - 웹페이지를 정보로써 더 의미있게 전달하기 때문이다
    - 단락과 단락 사이 간격이 고정되어 있어 시각적 자유도가 떨어지나, css를 사용하여 조정할 수 있다
    ex) <p style="margin-top:45px">
  6. <img>
    - 이미지 태그
    - src(sorce) 속성을 같이 사용해야한다
    - 인터넷 이미지 주소나 로컬 이미지 주소를 넣는다
    ex) <img src="https://s3-ap-northeast-2.amazonaws.com/opentutorials-user-file/module/3135/7648.png">
    - width 속성을 사용하여 숫자나 %로 크기를 조정할 수 있다
  7. <li>
    - 리스트 태그 - <ul>(unordered list), <ol>(ordered list)와 함께 사용한다
    ex)
    <ul>
      <li>
       <something 1
       <something 2
      </li>
    </ul>
  8. <a>
    - 링크 태그 (anchor)
    - ex) <a href="링크 주소" target="_blank" title="~~"> ㄴ href(HyperText Reference)
    ㄴ target="_blank" : 링크 클릭 시, 새 창에서 페이지가 열리게 하는 속성
    ㄴ title="~~" : 커서 올렸을 때, 링크가 어떤 내용을 담고 있는지 툴팁으로 보여주는 기능
    예시 링크(웹)
    (생코)
